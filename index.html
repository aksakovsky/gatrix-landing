The Problem

Hardware video matrices are holding your projects back

Physical matrices cost tens of thousands, lock you into fixed port counts, and can't scale beyond cable length.

Prohibitive Cost

A 16×16 HDMI matrix starts at $5,000. Enterprise SDI systems exceed $50,000. Budgets are consumed before the project even begins.

Rigid & Inflexible

Fixed port counts mean you buy for peak capacity. Need one more input? Buy an entirely new chassis. Scaling is expensive and wasteful.

Cable-Length Limited

HDMI reaches 15 meters. SDI reaches 100 meters. Distributed buildings? Multi-site deployments? Physical cables simply don't work.

Features

Everything you need to route video over IP

A complete software-defined video matrix with the tools AV professionals expect.

Multi-Protocol Routing

Translate between RTSP, NDI, SRT, WebRTC, HLS, IPMX, ONVIF and WHIP/WHEP seamlessly. Mix protocols in a single matrix — no adapters needed.

Drag-and-Drop Matrix

Visual grid interface for routing. Click to connect any source to any destination. Save and recall presets instantly.

Sub-200ms Latency

Hardware-accelerated encoding (NVENC, QSV, VideoToolbox) with low-latency presets. Real-time enough for live production.

Real-Time Monitoring

Track bitrate, FPS, latency and packet loss per channel. Instant alerts when a source drops or quality degrades.

REST API & Webhooks

Automate routing changes via API. Integrate with your control system, VMS, or scheduling software. Full programmability.

Auto-Discovery

Automatically finds cameras, encoders and NDI devices on your network via mDNS and SSDP. Zero manual configuration.

Presets & Scenes

Save entire routing configurations as presets. Switch between meeting mode, presentation mode, or recording mode with one click.

Failover & Redundancy

Automatic fallback streams. Watchdog process ensures 24/7 uptime. Stateless architecture for easy recovery.

Secure by Default

SRTP encryption, SRT passphrase protection. Role-based access control for multi-user environments. API authentication built in.

System Architecture

One Core. Infinite endpoints.

A central GPU-accelerated Core handles routing and transcoding. Lightweight Nodes extend to any PC or display. IP devices connect directly — zero extra hardware.

Gatrix achieves sub-200ms glass-to-glass latency for IP streams. For most broadcast, live production, and corporate AV workflows this is indistinguishable from hardware. Gatrix supports 13 protocols including RTSP, NDI, SRT, WebRTC, RTMP, HLS, and MPEG-TS. If your cameras or encoders output IP video, Gatrix can ingest and route them. It also handles protocol conversion — receive as SRT, deliver as NDI, for example — without extra transcoders.

Gatrix supports active-standby clustering. Deploy two instances and if the primary fails, the standby takes over automatically with your routing table intact. You can also run on redundant VMs or containers for the same resilience you'd expect from hardware — at a fraction of the cost.

With hardware, you're limited by backplane slots and frame size. Gatrix scales horizontally — need more I/O? Add another instance. Go from a 4×4 to a 64×64 matrix by upgrading your license, not your rack. No truck rolls, no downtime for card swaps.
